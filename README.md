### ✨기능

1. 구체 클릭 시 카운트 증가
2. 클릭하지 않아도 2초 마다 자동으로 카운트 증가
3. 상점 - 카운트로 구매
4. 업그레이드 - 클릭: 한 번의 클릭 시, 올라가는 카운트 증가
             - 자동: 자동으로 올라가는 카운트 증가
